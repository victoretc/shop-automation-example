# Стиль тестов

Не должно быть:
1. if, else в тестах
2. try except без необходимости 
3. time.sleep()
4. Не data-testid
5. Комментариев от LLm-ок

Не рекомендуется: 
1. wait_for()

Должно быть: 
1. data-testid для локаторов

# Работа с git 

Коммиты
test: <test name> - добавили тест
fix: <test name> - что то исправили
feat: <что добавлено> - фича фреймворка автотестов
doc: <о чем> - документация

М-ры
<номер задачи в jira>: <краткое описание>
